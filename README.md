### Environment requirement
> NodeJS: `v16`
>
> NPM: `v8`

## Installation

With yarn:

```bash
$ yarn install
```

With NPM:

```bash
$ npm install
```

### Configuration

-- config in file src/config/lcl.json

### Run the App

Once you have your setup ready, run:

With yarn:

    $ yarn dev

With npm:

    $ npm run dev
